# Image Uploader REST API
A simple REST API for uploading image

## 1. Requirements
1. Apache/NGINX with PHP5
2. MySQL

## 2. Installation
1. Clone this repo

## 3. HTTP Request
### Method
POST
### Content-Type
multipart/form-data
### Parameters
* k = test
* image_file = **User Input**
